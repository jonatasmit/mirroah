# PRD - Mirroah Deck Bar Website

## Data de Criação
14/02/2025

## Problema Original
Criar um website impressionante para o Mirroah Deck Bar beach club usando os itens do menu fornecidos, onde "Eventos" e "Contato" são botões para WhatsApp +5521972232170 com mensagens personalizadas. Site em português, clima tropical do Rio de Janeiro, tema carnaval o ano todo, com destaque para eventos de carnaval. Remover badge da Emergent e otimizar SEO para captar público do Faro Beach Club e Rio Beach Club.

## Informações de Contato
- **WhatsApp:** +5521972232170
- **Email:** contato@mirroahdeckbar.com.br
- **Instagram:** https://www.instagram.com/mirroahdeck/

## Arquitetura & Stack
- **Frontend:** React + Tailwind CSS + Shadcn UI
- **Tipo:** Landing Page (apenas frontend, sem backend necessário nesta fase)
- **Fontes:** Poppins (corpo) e Righteous (títulos)
- **Paleta de Cores:** Amarelo vibrante (#FDB827), Rosa/Magenta (#FF006E), Laranja coral (#FF6B35), Turquesa (#06D6A0)
